What data does safesoundr store?

safesoundr stores the following data in the cloud — all of which is fully anonymous:
  • A randomly generated anonymous User ID (UUID)
  • Noise incident location (latitude & longitude)
  • Date, time, and noise type of each logged incident
  • Disturbance severity score

No personal information (name, email, phone, address, or audio recordings) is ever stored in the cloud.

To request deletion of all anonymous data associated with your device, please email us using the button below. Include the subject line "Data Deletion Request" and we will process your request within 7 days.
